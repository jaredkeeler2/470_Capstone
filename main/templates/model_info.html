Model Information

Validation

Each model's accuracy is displayed via the Mean Absolute Error (MAEA MAE closer to 0 is ideal. The higher the number, the less accurate the prediction.). To evaluate the models we trained on all but the last two terms, then forecasted the last two terms. We displayed those predictions as the validation points. We then calculated the MAE by averaging the difference between the predictions and the actual enrollments.

ARIMA

Definition: An ARIMA (AutoRegressive Integrated Moving Average) model is a statistical technique for forecasting time series data, using its historical values to predict future ones. ℹ️ Hover for model info Our model used historical data. We trained on all terms except the last two and forecasted those two for validation.

ARIMAX

Definition: An ARIMAX model is an Autoregressive Integrated Moving Average with Explanatory Variables, a statistical model used for time series forecasting. ℹ️ Hover for model info Our model used historical data along with highschool data and course prerequisites as exogenous variables. Highschool data was only included for courses below and including 201. For courses above 201, we only used prerequisites. We trained on all terms except the last two and forecasted those two for validation.

https://www.statsmodels.org/stable/generated/statsmodels.tsa.statespace.sarimax.SARIMAX.html

SARIMA

Definition: A SARIMA model is a time series forecasting model that accounts for both non-seasonal and seasonal patterns by combining autoregressive (AR), integrated (I), and moving average (MA) components. ℹ️ Hover for model info Our model used a seasonal period of m=3 for courses below and including 201 and m=2 for courses above 201. We trained on all terms except the last two and forecasted those two for validation.

Model Information

Validation

Each model's accuracy is displayed via the Mean Absolute Error (MAE). To evaluate the models we
trained on all but the last two terms, then forecasted the last two terms. We displayed those predictions as
the validation points. We then calculated the MAE by averaging the discrepancy between the predictions
and the actual enrollments.

ARIMAX

Definition: An ARIMAX model is an Autoregressive Integrated Moving Average with Explanatory Variables, a statistical model used for time series forecasting.

SARIMA

Definition: A SARIMA model is a time series forecasting model that accounts for both non-seasonal and seasonal patterns by combining autoregressive (AR), integrated (I), and moving average (MA) components.

https://www.statsmodels.org/stable/generated/statsmodels.tsa.statespace.sarimax.SARIMAX.html

SARIMAX

Definition: A SARIMAX model is a time series forecasting model that combines the Seasonal Autoregressive Integrated Moving Average (SARIMA) model with Exogenous variables.
